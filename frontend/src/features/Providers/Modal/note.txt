openModal usage:
1. when using updating content then you need to make an own component with api calls within that component and call that component with given data
2. when using a not updating content you can just use anything you want

examples:
for point 1(Updating content): 
1. Component SerieModalContent call and passing the series and type var)
openModal(<SerieModalContent selectedSeries={series} type={type!} />, ModalTypes.SMALLBOX);


for point 2(Static content):
1. using html code within the openModal
  const handleOpenModal = () => {
    openModal(() => (
      <div>
        <h2 style={{ color: 'green' }}>Logout Content</h2>
